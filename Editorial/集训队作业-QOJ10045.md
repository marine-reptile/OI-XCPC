#### QOJ 10045

##### 题目来源

[Petrozavodsk Summer 2024. Day 5: Der Kontest J题](https://qoj.ac/contest/1902/problem/10045)

##### 题目描述

有 $k$ 个长度为 $n$ 的排列，然后，我们将这些排列的逆排列也写入，这样我们就有了 $2k$ 个排列。

然后我们在 $2k$ 行中每行写下一个排列，得到一个 $(2k)\times n$ 的矩阵。

最后，这个矩阵的每一列内的所有元素将被随机排列，得到一个新的矩阵。

请根据这个最后的矩阵，构造一组可能的初始的 $k$ 个排列，可以输出任意一组解。

逆排列：我们称排列 $b$ 是排列 $p$ 的逆排列，当且仅当对任意 $1\le i\le n$，都有 $b_{p_i}=i$。

##### 输入格式

第一行两个整数 $n,k$，表示排列的长度和初始排列的个数。

接下来 $2k$ 行每行 $n$ 个数，描述所有操作后的最终矩阵。

##### 输出格式

$k$ 行每行 $n$ 个整数，表示所有的排列（它们的逆排列无需输出）

##### 输入输出样例 #1

###### 输入 #1

```
3 1
1 2 3
1 2 3
```

###### 输出 #1

```
1 2 3
```

##### 输入输出样例 #2

###### 输入 #2

```
4 2
1 1 3 4
4 3 2 1
2 4 2 4
1 3 3 2
```

###### 输出 #2

```
1 3 2 4
4 1 3 2
```

##### 说明/提示

满足 $n\le 4\times 10^4$，$k\le 7$，保证一定存在一组合法的解。

##### 题目解析

我们考虑排列和逆排列存在什么性质，发现它们满足：排列的置换环是它的逆排列的置换环的反图。

那么我们考虑这样一个思路：我们对于当前的矩阵，每次找到一个排列和它的逆排列，然后将它们从中删除。这等价于，我们将所有的置换环拼到同一个图上，然后每次找到一组置换环，使得它上面的每条边都存在正反两个方向，且不重不漏的覆盖了所有点，然后将它们从图上删除。

我们发现，这样求答案一定是可行的，我们首先把一正一反两条边写作一条无向边，然后因为每次取出一组这样的置换环会消耗一个点的两个度数，而每个点的总度数一定是 $2k$。也就是，在 $k-1$ 次查询之后，每个点一定恰剩下两个度数，此时剩下的图一定仍然是个置换环。

然后考虑如何求所有排列的置换环图的并，发现我们对矩阵中的每个 $a_{i,j}=x$，连一条 $(j\rightarrow x)$ 的有向边即可。这是因为在重排列之前，我们连边只需要对所有 $j\rightarrow x$ 连边，而重排后其实只改变了每个元素的 $i$，对 $j,x$ 没有任何改变。

然后考虑怎么每次取出一组置换环，发现如果我们再给这张无向图重定向，使得每条边重新变成一个单向置换环，那么我们可以使用二分图匹配跑出一组出度到入度的完美匹配，就可以保证覆盖每个点。

对无向图的重定向可以用欧拉回路求取，二分图的完美匹配可以每次使用网络流算法求取。一共进行 $k$ 次，每次在一张 $2n$ 个点 $2nk$ 条边的图上求一组匹配，复杂度 $O(k^2n\sqrt n)$。
